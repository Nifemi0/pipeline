Capabilities
🏗️

Blue-Collar Focus

Targets plumbers, electricians, roofers, contractors, and other trades — the businesses most likely to need a website.

📡

Multi-Signal Scoring

Phone, address, email, Facebook presence, and website detection — scores each lead hot/warm/cold automatically.

🤖

AI-Powered Pitches

Each pitch is personalized using Gemini AI — references the business name, category, and location.

🔁

Fully Autonomous

Scout → Analyze → Write → Track. The pipeline runs on a cron schedule and surfaces results in real-time.

How To Use
01

Explore the Dashboard

Click "Enter Mission Control" to see your pipeline overview — total leads, hot/warm/cold breakdown, and agent status.

02

Browse & Filter Leads

Use the Leads page to search by city, category, or score. Click any lead to see its full analysis and pitch history.

03

Run Agents on Demand

Go to Controls — run the Analyst to score pending leads, the Writer to generate pitches, or the full Pipeline for a complete pass.

04

Track Results

The Analytics page shows score distribution, top cities, top categories, and daily activity — everything you need to monitor performance.
